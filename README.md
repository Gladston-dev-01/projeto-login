# 🔐 Tela de Login Moderna e Responsiva

Este projeto consiste em uma interface de autenticação (Tela de Login) moderna, limpa e responsiva, desenvolvida com **HTML5** e **CSS3**. O foco principal foi criar um formulário visualmente atraente, com foco em usabilidade (UX) e acessibilidade no preenchimento de dados.

---

## 🛠️ Tecnologias e Conceitos Praticados:

* **Elementos de Formulário:** Uso semântico de tags como `<form>`, `<input>`, `<label>` e tratamento de tipos de dados (e-mail, password).
* **Design Responsivo:** Layout flexível que se adapta perfeitamente a computadores, tablets e telas de smartphones.
* **Estilização Avançada:** Uso de sombras (`box-shadow`), cantos arredondados, gradientes de fundo e efeitos visuais nos campos de foco (`:focus`).
* **Inclusão de Ícones:** Integração de ícones visuais para melhorar a identificação dos campos de usuário e senha.
